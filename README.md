# DSS-Catedra
Este proyecto busca desarrollar un sistema que facilite la gestión de empleados, clientes, productos financieros, transacciones y usuarios de la cooperativa "ACOEMPRENDEDORES". Además, incluirá una cartera virtual para que los clientes consulten sus productos y realicen operaciones de manera eficiente.
